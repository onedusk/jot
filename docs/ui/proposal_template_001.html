Abel Hydro Documentation

Abel Hydro is a distributed IoT system for hydroponic monitoring and control. This documentation covers the complete system architecture, hardware specifications, and software implementation.

Overview

The Abel Hydro system provides comprehensive monitoring and control capabilities for hydroponic installations through a distributed mesh network of ESP32-based nodes. The system is designed for scalability, reliability, and ease of deployment.

Key Features

  • Distributed mesh network architecture
  • Real-time sensor monitoring
  • Automated control systems
  • Cloud connectivity and remote management
  • Ultra-low power consumption design

System Architecture

The system employs a hierarchical architecture with three primary node types: sensor nodes for environmental monitoring, actuator nodes for system control, and a gateway controller for coordination and cloud connectivity.

Note: The distributed architecture ensures system resilience through redundancy and automatic failover capabilities.

Code Example

// Initialize sensor node
void setup() {
    Serial.begin(115200);
    
    // Initialize mesh network
    mesh.init(MESH_PREFIX, MESH_PASSWORD, &userScheduler, MESH_PORT);
    mesh.onReceive(&receivedCallback);
    mesh.onNewConnection(&newConnectionCallback);
    
    // Initialize sensors
    initSensors();
}

Component Specification Purpose

Sensor MCU ESP32-C3 Sensor and actuator nodes
Power Supply 5V/3.3V regulated System power
Communication WiFi/BLE/ESP-NOW Mesh networking
